### computed

```js
// 写法1
<div>{{c}}</div>
data {
  a:1,
  b:2
},
  computed: {
    c() {
      return this.a+this.b
    }
  }
// 写法2
<div>{{add()}}</div>
data {
  a:1,
  b:2
},
  methods: {
    add() {
      return this.a+this.b
    }
  }
// 结果都是 a或者b改变时 页面显示的值会发生变化
// computed 会做缓存
```
