### 安装npm包常见问题

1. 安装decimal包的时候报错 ： 

```js
error decimal@0.0.2: The engine "node" is incompatible with this module. Expected version "0.4.x". Got "10.13.0"
error Found incompatible module.
```

解决方案： `yarn add --ignore-engines decimal ` 忽略版本
